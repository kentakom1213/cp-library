# cp-library

## cp-library-rs

競技プログラミング用のライブラリ

ドキュメント：https://kentakom1213.github.io/cp-library/cp_library_rs/

## expander

以下のエイリアスを設定します．

```
alias bundle='<clone先のパス>/cp-library/expander/target/release/expand'
```

### 展開

```sh
$ bundle <展開するRustファイル>
```

### 復元

```sh
$ bundle <復元するRustファイルのパス> -r
```
